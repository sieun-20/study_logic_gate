### 1) 불대수
- 오직 참(1)과 거짓(0) 두가지의 결과 값만을 가지는 대수.
- 신호가 통할 때는 1이라고 표현하고(HIGH), 신호가 통하지 않을 때는 0이라 표현(LOW) (통하지 않는 0 신호를 1호 바꾸는 작업 가능)
## 불대수의 기본 법칙
- 기본 법칙
  1. A+0=A
  2. A·1=A
  3. A+1=A
  4. A·0=A
  5. A+A=A
  6. A·A=A
- 보수성의 법칙
  1. A+A'=1
  2. A·A'=0
  3. A''=A
- 교환 법칙
  1. A+B=B+A
  2. A·B=B·A
- 결합 법칙
  1. (A+B)+C=A+(B+C)
  2. (A·B)·C=A·(B·C)
- 분배 법칙
  1. A·(B+C)=A·B+A·C
  2. A+B·C=(A+B)·(A+C)
- 흡수 법칙
  1. A+A·B=A
  2. A·(A+B)=A
- 드모르간의 정리
  1. A'
